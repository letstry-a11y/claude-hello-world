## 程序简介

动物世界动画是一个基于 Python tkinter 开发的互动动画程序。在这个虚拟的动物世界中，你可以：
- 添加 8 种不同的可爱动物
- 观看动物在草地上自由移动和互动
- 切换不同的天气效果
- 观察动物之间的碰撞反应

---

## 系统要求

- **操作系统**: Windows 7 及以上
- **Python 版本**: Python 3.x
- **依赖库**: tkinter (Python 标准库，无需额外安装)
- **屏幕分辨率**: 建议 1024x768 或更高

## 功能使用指南

### 1. 添加动物

**步骤：**
1. 点击"选择动物"下拉菜单
2. 选择你想要的动物：
   - 随机（随机添加一种动物）
   - 大象
   - 狮子
   - 长颈鹿
   - 猴子
   - 熊猫
   - 老虎
   - 兔子
   - 斑马
3. 点击"添加动物"按钮

**说明：**
- 每次点击添加一只动物
- 动物会随机出现在地面区域的不同位置
- 可以添加多个相同种类的动物
- 建议添加 5-10 只动物以获得最佳观赏效果

### 2. 控制动画

#### 开始动画
- 点击"开始动画"按钮
- 动物开始左右移动
- 云朵开始缓慢飘动
- 天气效果（雨、雪）开始动画

#### 停止动画
- 点击"停止动画"按钮
- 所有动画暂停
- 可以继续添加动物
- 再次点击"开始动画"继续

### 3. 切换天气

#### 晴天模式（默认）
- 明亮的浅蓝色天空
- 太阳在右上角
- 白云缓缓飘动
- 适合清晰观察动物

#### 雨天模式
- 天空变为浅灰色
- 50 条蓝色雨滴从上往下落
- 雨滴到达底部后自动回到顶部
- 营造雨天氛围

#### 雪天模式
- 天空变为淡钢蓝色
- 40 片白色雪花飘落
- 雪花左右摆动，模拟真实飘雪
- 创造冬季场景

**提示：** 可以在动画运行时随时切换天气！

### 4. 观察动物行为

#### 基本移动
- 动物会以不同速度左右移动
- 碰到边界会自动转向
- 每只动物的速度随机（1-2 像素/帧）

#### 随机跳跃
- 动物有 2% 的概率随机跳跃
- 跳跃时会短暂离地
- 200 毫秒后落地

#### 碰撞反应
- 当两只动物靠近（距离 < 40 像素）时触发
- 双方交换移动方向
- 同时产生小幅跳跃效果
- 模拟真实的碰撞反弹

## 常见问题

### Q1: 程序无法启动怎么办？
**A:**
1. 确认已安装 Python 3.x
2. 检查 `run_animation.bat` 中的 Python 路径是否正确
3. 尝试在命令行中直接运行：`python animal_world_animation.py`
4. 查看是否有错误信息

### Q2: 动物移动太快或太慢？
**A:** 动物的速度是随机的，范围在 1-2 像素/帧。如果想调整速度：
- 打开 `animal_world_animation.py` 文件
- 找到 `'dx': random.choice([-2, -1, 1, 2])` 这一行
- 修改数值即可（如 `[-3, -2, 2, 3]` 会更快）

### Q3: 能添加多少只动物？
**A:** 理论上没有上限，但建议：
- 5-10 只：最佳观赏效果
- 10-20 只：热闹但不拥挤
- 20 只以上：可能影响性能

### Q4: 如何清除所有动物？
**A:** 当前版本需要重启程序。未来版本可能添加"清除全部"按钮。

### Q5: 天气效果不明显？
**A:**
- 确保已点击"开始动画"按钮
- 雨天和雪天的效果需要动画运行才能看到
- 可以尝试切换到其他天气再切换回来

### Q6: 动物碰撞后重叠怎么办？
**A:** 这是正常现象。碰撞检测使用简单的距离算法，偶尔会有重叠情况，但会自动分开。

### Q7: 窗口大小能调整吗？
**A:** 当前窗口大小固定为 800x600 像素。如需修改：
- 打开 `animal_world_animation.py`
- 修改第 12 行：`self.root.geometry("800x600")`
- 注意：需同步调整画布和其他元素尺寸

### Q8: 可以添加音效吗？
**A:** 当前版本不支持音效。如需添加音效，需要：
1. 安装 pygame 或 playsound 库
2. 准备音频文件（.wav 或 .mp3）
3. 修改代码添加音效播放功能

## 使用技巧

### 技巧 1：创建动物派对
1. 添加 10-15 只不同种类的动物
2. 点击"开始动画"
3. 观察动物之间的互动和碰撞

### 技巧 2：天气变化观赏
1. 先添加 5-8 只动物
2. 开始动画
3. 每隔 10 秒切换一次天气
4. 体验不同天气下的动物世界

### 技巧 3：特定动物群
- 只添加大型动物（大象、长颈鹿）观察稳重移动
- 只添加小型动物（兔子、猴子）观察灵活互动
- 添加同类动物（如 8 只熊猫）创造专属场景

### 技巧 4：慢动作观察
如果想慢速观看动物行为：
1. 打开 `animal_world_animation.py`
2. 找到最后一行：`self.root.after(50, self.animate)`
3. 将 50 改为更大的数字（如 100），动画会变慢
